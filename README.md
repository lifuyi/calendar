# 状态栏日历应用

一个macOS状态栏应用程序，显示当前日期、星期和时间，点击可展开显示完整日历视图。

## 功能特点

- 在状态栏实时显示当前日期时间（格式：MM月dd日 HH:mm E）
- 点击状态栏可展开日历视图
- 支持年份选择（1900-2075年）
- 支持月份切换
- 支持返回今天
- 显示农历日期
- 标记中国法定节假日和节气
- 显示当前天气信息（温度、天气状况、湿度、气压、降水量、体感温度）
- 显示当天日历事件（需要用户授权）

## 系统要求

- macOS 12.0 或更高版本
- Xcode 13.0 或更高版本（用于开发）

## 安装和运行

### 命令行构建和运行

```bash
# 克隆项目
git clone [项目地址]
cd calendar

# 构建项目
swift build

# 运行项目
swift run
```

### Xcode 构建和运行

1. 打开 Xcode
2. 选择 "Open a project or file"
3. 选择项目目录中的 `Package.swift` 文件
4. 点击 "Open"
5. 点击 "Run" 按钮或按 `Cmd+R`
