# 状态栏日历应用

一个macOS状态栏应用程序，显示当前日期、星期和时间，点击可展开显示完整日历视图。

## 功能特点

- 在状态栏显示当前日期、星期和时间
- 点击状态栏可展开日历视图
- 支持年份选择（1900-2075年）
- 支持月份切换
- 支持返回今天
- **显示农历日期**
- **标记法定节假日和节气**
- **显示当前天气信息（温度、天气状况、湿度、气压、降水量、体感温度）**

## 系统要求

- macOS 12.0 或更高版本
- Xcode 13.0 或更高版本（用于开发）

## 安装和运行

1. 克隆项目到本地：
```bash
git clone [项目地址]
cd calendar
```

2. 构建和运行项目：
```bash
swift build
swift run
```
